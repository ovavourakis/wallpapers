# Binary Clock Wallpaper

This is a binary clock wallpaper that displays time and date information as binary numbers using white circles.

## Display Format

- **Minute** (0-59): 6 binary digits (circles)
- **Hour** (0-23): 5 binary digits
- **Date** (1-31): 5 binary digits
- **Month** (1-12): 4 binary digits

Filled white circles represent 1s, empty circles (with white outline) represent 0s.

## Installation

This is just a local webpage in HTML. Use an app like Plash to display it over the desktop.
